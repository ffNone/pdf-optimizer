# برنامج منشئ ملفات PDF

برنامج ويب متكامل وReady للاستخدام يسمح بإنشاء ملفات PDF تحتوي على نصوص وصور بدعم كامل للغة العربية.

## المميزات

✅ **دعم اللغة العربية بالكامل** - النص والواجهة بالعربية
✅ **Responsive Design** - يعمل على جميع الأجهزة (ديسكتوب، تابلت، موبايل)
✅ **معاينة الصور** - معاينة فورية للصور المختارة
✅ **حذف الصور** - إمكانية حذف الصور قبل إنشاء PDF
✅ **تصميم عصري** - واجهة جذابة وسهلة الاستخدام
✅ **دعم النصوص الطويلة** - textarea كبير لاستيعاب نصوص طويلة

## الملفات المضمنة

- `index.html` - الواجهة الرئيسية
- `styles.css` - أنماط الموقع (Responsive)
- `script.js` - الوظائف الرئيسية

## كيفية الاستخدام

1. افتح ملف `index.html` في المتصفح
2. أدخل اسم الملف المراد حفظه
3. أدخل النص في حقل النص
4. اختر الصور المراد إضافتها (اختياري)
5. انقر على زر "تحميل PDF"

## مواصفات الملف الناتج

- **خط النص**: Bold وواضح جداً
- **حجم النص**: 26px
- **الصور**: موضوعة تحت النص بمسافة واحدة، جنب بعضها وواضحة
- **الدعم**: اللغة العربية بشكل كامل
- **الصيغة**: PDF قابلة للطباعة والمشاركة

## المتطلبات

- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- اتصال إنترنت (لتحميل مكتبة html2pdf من CDN)

## الملاحظات التقنية

- البرنامج يستخدم مكتبة `html2pdf.js` المجانية
- جميع العمليات تتم على جهاز المستخدم (لا يتم إرسال بيانات إلى الخادم)
- الصيغ المدعومة للصور: JPG, PNG, GIF, WebP
- يمكن إضافة عدد غير محدود من الصور

## نصائح للاستخدام الأمثل

- استخدم صور واضحة وعالية الجودة
- تأكد من أن النص مكتوب بشكل صحيح قبل إنشاء PDF
- إذا كان لديك صور كثيرة، قد يستغرق إنشاء PDF وقتاً أطول قليلاً
